# Kerala Reiseplanung: Die ultimative Checkliste

Von e-Visa-Antrag bis Moskitonetz: Diese Checkliste führt Sie Schritt für Schritt durch die Vorbereitung. Unser Team in Kerala hilft bei Transfers und Hotelbuchungen.

## 3-6 Monate vor der Reise

### Reiseplanung
- [ ] **Reisezeit festlegen** (November-März optimal)
- [ ] **Reisedauer bestimmen** (mindestens 7 Tage)
- [ ] **Reiseart wählen** (Ayurveda, Rundreise, Strand)
- [ ] **Budget planen** (Flug, Unterkunft, Aktivitäten)
- [ ] **Reiseversicherung** abschließen

### Dokumente
- [ ] **Reisepass prüfen** (mindestens 6 Monate gültig)
- [ ] **E-Visum beantragen** (online, 30 Tage gültig)
- [ ] **Impfpass aktualisieren** (Hepatitis A/B, Typhus)
- [ ] **Internationaler Führerschein** (falls Mietwagen)

### Gesundheit
- [ ] **Hausarzt konsultieren** (besonders bei Ayurveda)
- [ ] **Impfungen auffrischen** (Hepatitis A/B, Typhus)
- [ ] **Malaria-Prophylaxe** (in Monsunzeit)
- [ ] **Reiseapotheke** zusammenstellen
- [ ] **Gesundheitszeugnis** (für Ayurveda-Kuren)

## 1-3 Monate vor der Reise

### Buchungen
- [ ] **Flug buchen** (früh buchen = günstiger)
- [ ] **Unterkunft reservieren** (Resorts, Hotels)
- [ ] **Transfers organisieren** (Flughafen, zwischen Orten)
- [ ] **Aktivitäten buchen** (Hausboot, Safari, Ayurveda)
- [ ] **Reiseführer** bestellen oder herunterladen

### Finanzen
- [ ] **Kreditkarte** für Reisen freischalten
- [ ] **Bargeld** in Euro mitnehmen (für Wechsel)
- [ ] **Reiseschecks** (optional, aber sicher)
- [ ] **Bank informieren** über Reise
- [ ] **Wechselkurs** aktuell halten

### Kommunikation
- [ ] **Handy entsperren** für internationale Nutzung
- [ ] **Roaming-Optionen** prüfen
- [ ] **WhatsApp** installieren (sehr beliebt in Indien)
- [ ] **Offline-Karten** herunterladen
- [ ] **Wichtige Kontakte** notieren

## 2-4 Wochen vor der Reise

### Packliste
- [ ] **Kleidung** (leichte Baumwolle, lange Hosen)
- [ ] **Schuhe** (bequeme, geschlossene Schuhe)
- [ ] **Sonnenbrille** und **Sonnenhut**
- [ ] **Sonnencreme** (hoher Lichtschutzfaktor)
- [ ] **Insektenschutz** (Mückenspray)
- [ ] **Erste-Hilfe-Set** (Pflaster, Desinfektionsmittel)
- [ ] **Medikamente** (persönliche Medikamente)
- [ ] **Kamera** und **Ladegeräte**
- [ ] **Adapter** für Steckdosen (Typ D/M)

### Spezielle Ausrüstung
- [ ] **Moskitonetz** (für einfache Unterkünfte)
- [ ] **Taschenlampe** (für Stromausfälle)
- [ ] **Wasserdichte Taschen** (für Regenzeit)
- [ ] **Handtücher** (schnelltrocknend)
- [ ] **Toilettenpapier** (nicht überall verfügbar)

### Vorbereitung
- [ ] **Reiseführer lesen** über Kerala
- [ ] **Kulturelle Regeln** lernen
- [ ] **Grundlegende Hindi-Wörter** lernen
- [ ] **Wettervorhersage** prüfen
- [ ] **Check-in online** (24h vor Abflug)

## 1 Woche vor der Reise

### Finale Vorbereitung
- [ ] **Gepäck packen** (Gewicht prüfen)
- [ ] **Wichtige Dokumente** kopieren
- [ ] **Kopien** an Familie/Freunde senden
- [ ] **Haustiere** versorgen
- [ ] **Post** umleiten
- [ ] **Zeitungen** abbestellen

### Gesundheit
- [ ] **Letzte Impfungen** (falls nötig)
- [ ] **Medikamente** für Reise packen
- [ ] **Gesundheitszeugnis** mitnehmen
- [ ] **Krankenversicherung** prüfen
- [ ] **Notfallkontakte** notieren

## Am Reisetag

### Vor dem Abflug
- [ ] **Dokumente** griffbereit haben
- [ ] **Gepäck** nochmals prüfen
- [ ] **Handgepäck** vorbereiten
- [ ] **Flughafen** rechtzeitig erreichen
- [ ] **Check-in** und **Security**

### Während des Flugs
- [ ] **Ausfüllformulare** für Einreise
- [ ] **Zeitverschiebung** beachten
- [ ] **Ausreichend trinken**
- [ ] **Bewegung** (gegen Thrombose)

## In Kerala

### Ankunft
- [ ] **Flughafen** verlassen
- [ ] **Transfer** zum Hotel
- [ ] **Geld wechseln** (Flughafen oder Hotel)
- [ ] **SIM-Karte** kaufen (optional)
- [ ] **Hotel** einchecken

### Erste Tage
- [ ] **Jetlag** auskurieren
- [ ] **Lokale Gewohnheiten** beobachten
- [ ] **Sicherheitsregeln** beachten
- [ ] **Wasser** nur abgefüllt trinken
- [ ] **Straßenverkehr** vorsichtig sein

## Praktische Tipps

### Geld
- **ATM-Gebühren** beachten
- **Kreditkarten** nicht überall akzeptiert
- **Trinkgeld** üblich (10-15%)
- **Preise verhandeln** auf Märkten

### Sicherheit
- **Wertsachen** im Hotelsafe
- **Kopien** von Dokumenten mitnehmen
- **Nachts** nicht allein unterwegs
- **Offene Kleidung** vermeiden

### Gesundheit
- **Nur abgefülltes Wasser** trinken
- **Rohes Gemüse** vermeiden
- **Hände** regelmäßig waschen
- **Sonnenschutz** nicht vergessen

## Fazit

Mit dieser Checkliste sind Sie optimal auf Ihre Kerala-Reise vorbereitet. Denken Sie daran: Es ist besser, zu viel vorbereitet zu sein als zu wenig!
